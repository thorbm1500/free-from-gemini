# Free From Gemini (FFG)
Too much AI is being forced upon users of various platforms.
One of these platforms is Google Drive with the services like Google Docs, Google Sheets, etc.<br>

Humans are all unique and different, and no one is to decide whether someone should like AI or not.
This should go both ways, but the people that dislike AI are sadly being ignored.<br>
#### Using AI should be a choice, and not an obligation.
